# Tasklists Workflows

Tasklists group related tasks.

## Create a task list

```bash
lark tasklists create --name "Backlog"
```

## Get task list details

```bash
lark tasklists info <TASKLIST_GUID>
```

## Update a task list

```bash
lark tasklists update <TASKLIST_GUID> --name "Team Backlog"
```

## Delete a task list

```bash
lark tasklists delete <TASKLIST_GUID>
```
